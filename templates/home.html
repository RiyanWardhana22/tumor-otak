Sistem Deteksi Tumor Otak

Sistem berbasis web untuk membantu identifikasi tumor otak melalui citra MRI dengan model CNN MobileNetV2.

Tentang Sistem

Sistem ini dikembangkan menggunakan metode Convolutional Neural Network (CNN) berbasis MobileNetV2 untuk mengklasifikasi citra MRI otak menjadi beberapa jenis tumor.

  • Dataset: Brain Tumor MRI Dataset dari Kaggle (4 kelas)
  • Preprocessing: Resize, Normalisasi, Augmentasi
  • Evaluasi: Akurasi, Presisi, Recall

Apa itu CNN?

Convolutional Neural Network (CNN) adalah algoritma Deep Learning untuk analisis visual yang mampu mengenali pola kompleks dalam gambar.

Mengapa MobileNetV2?

MobileNetV2 ringan, cepat, dan efisien untuk aplikasi web tanpa mengorbankan akurasi, ideal untuk klasifikasi citra medis seperti MRI.
